Additions to game engine
  • Pause/Resume Option - Allows the player to pause the game and resume
  • Simple Maze Generator - For the game shown it generated a 100x100 size maze with 16 doors (8 links).
  • Zoomed In/Image Tile - The game only displays an area of 5 x 3 squares, instead of the full 100x100 square maze, of where the character is in relation to the maze. The image tiles are also generated based on the view of the area that the player has.
  • Side Scrolling (Updated) - The scroll background has been implemented for a grid based game. Previously, the side scrolling was implemented based on pixel by pixel movement as seen in Candy4Aliens. Now one or the other side scrolling background can be used.
